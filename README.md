# Projeto 2 de 'Fundamentos de computação concorrente, paralela e distribuida'.

O objetivo do projeto é praticar fundamentos de docker e arquitetura em microsserviços. Possuindo desafios envolvendo desde a criação até o gerenciamento de aplicações em containers.

## Estrutura

Conforme os requisitos, a estrutura do repositório é dividada em 5 pastas, uma para cada desafio. Cada pasta possui sua prórpia estrutura e seu próprio `readme` (incluindo as instruções de como executar e a explicação do código).

```text
projeto2-fccp\
├── desafio1\
├── desafio2\
├── desafio3\
├── desafio4\
├── desafio5\
├── .gitignore
├── LICENSE
└── README.md
```

> **OBS:** É necessário entrar na raiz de cada desafio para poder executar ele.
